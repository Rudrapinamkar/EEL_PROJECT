# EEL_PROJECT
🇮🇳 Indian Tourism & Statistics Dashboard

A data visualization project that explores tourism trends in India using structured datasets and interactive dashboards.
This project transforms raw tourism data into meaningful insights about travel patterns, economic impact, and regional popularity.

 Objective

To analyze and visualize tourism trends in India by focusing on:

Domestic Tourist Visits
Foreign Tourist Arrivals
Revenue Generated from Tourism
State-wise Tourism Distribution

The goal is to identify patterns, growth trends, and high-performing regions in India’s tourism sector.

 Methodology
Collected tourism datasets from reliable public sources
Cleaned and standardized inconsistent data formats
Handled missing values and removed irrelevant entries
Merged datasets into a unified structure using Python
Built an interactive dashboard for visualization
Generated reports and presentation for insights
